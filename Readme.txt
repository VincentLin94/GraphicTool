【需求】
	功耗/稳定性测试所得数据需要处理成折线图形式，方便观察问题

	现有方式在导出后需要手动转为表格，并通过Excel绘制成图

	工作重复性较高，故设计自动化工具

【实现】
	Python编写并打包成可执行文件

	可处理【cap-mem.txt】与【battery_info.log】两种常用log文件

	自动生成【电量曲线】和【内存曲线】或【电量曲线】和【运行电压&电流曲线】

【使用】
	将log文件导出至exe工具所在目录

	双击运行exe

	程序会自动创建子目录，并将当前log和生成的曲线图放在其内
	
	随后可放入新的log文件，重复运行

【FAQ】
	文件名限定为【cap-mem.txt】【cap_mem.txt】【battery_info.log】三者之一

	无需使用管理员模式运行，直接双击即可
